Storefront

A storefront web application that features a product grid displaying items retrieved from the FakeStoreAPI. Users can browse products, fetch more products on demand, view detailed descriptions through an accordion-style toggle, and navigate between pages using a functional navigation bar with pagination. A fully interactive shopping cart allows users to add products, adjust quantities, and view a dynamically updated total cost.

Members Only Message Board

A secure message board with user authentication implemented using Passport. Users can log in with verified identities, and their sessions remain active until the page is reloaded. For security, passwords are hashed before storage. Access to post authors and the ability to create posts is restricted and users must enter a secret member password to unlock these features. All user data and posts are stored in a PostgreSQL database, ensuring efficient data management and retrieval.

Memory Card Game

A memory card game featuring character images from Robohash. The objective is for users to click as many unique images as possible without repeating selections. The game dynamically tracks the current score and displays the session's high score. Each time a card is clicked, the cards are shuffled to increase difficulty, ensuring an engaging and challenging experience.

Resume Creator

A resume creator that allows users to input and format their resumes through interactive forms. Users can add personal information, work experience, and education, with the flexibility to edit or delete any entry. Additional work and educational experiences can be included, along with custom informational fields for each entry to provide further details. The application ensures a user-friendly experience for building, updating, and organizing professional resumes efficiently.

Pokemon Search

A pokedex that allows users to search for a Pokémon and retrieve detailed information from the PokeAPI. Upon entering a Pokémon's name or ID, the application fetches and displays relevant data, such as its type, abilities, stats, and sprite. The project provides a simple and interactive way to explore Pokémon details through real-time API integration.

Inventory Management

Pokemon trainer management app. Users can create, view, update, and delete individual trainers, as well as manage the Pokémon in their parties. Pokémon information is fetched from the PokeAPI and stored in a PostgreSQL database, ensuring that all details about each Pokémon are readily available. Trainers and Pokémon are maintained in separate tables, allowing for structured data organization and efficient retrieval. This system provides a practical example of relational database management while integrating external API data.
